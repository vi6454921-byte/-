# Среда — личный ИИ-ассистент Влада

Среда — это локально работающий ассистент, который слушает тебя голосом или текстом, управляет компьютером (Windows) и телефоном (Android через ADB), помнит факты о тебе, наблюдает за привычками, ведёт заметки и напоминания, считает статистику активности и ищет в интернете. Дизайн вдохновлён HUD-интерфейсом из вселенной Тони Старка — концентрические кольца, голографические индикаторы, тёплые акценты.

---

## ⚡ Быстрый старт

1. Установи Python 3.10+ (https://python.org).
2. Открой PowerShell в папке проекта и выполни:

   ```powershell
   cd sreda
   python -m pip install -r requirements.txt
   ```

3. (опционально) Установи Ollama https://ollama.com и `ollama pull mistral` — будет локальный fallback, если интернет отвалится.
4. Запусти двойным кликом `ЗАПУСТИТЬ_СРЕДУ.cmd` (или `START_SREDA.cmd`).
5. Откроется браузер → `http://localhost:5000`.

В трее запустится фоновый слушатель **Ctrl + Alt + S** — нажми где угодно, и Среда раскроется в безрамочном окне Chrome.

С телефона зайди по `http://<IP_твоего_ПК>:5000` (см. поле IP в левой панели). Должен быть один Wi-Fi.

## 🧠 Что Среда умеет (90 % того, что нужно каждый день)

Тебе **не нужно** учить никаких команд. Просто скажи, что хочешь — Среда сама разберёт намерение и сделает.

### 🖥 Управление ПК
- открыть/закрыть программу: «открой хром», «закрой VS Code», «запусти телегу»
- открыть сайт: «открой YouTube», «зайди на github»
- открыть папку: «покажи мне Загрузки»
- громкость: «сделай тише», «убавь до 30%», «mute»
- медиа: «поставь на паузу», «следующий трек»
- блокировка / выключение / перезагрузка / сон: «вырубись через час», «спать»
- буфер обмена: «скопируй сюда мой email», «вставь из буфера»
- набор текста и хоткеи: «нажми ctrl+shift+t»
- скриншот: «сделай скрин экрана» → картинка прилетит прямо в чат
- состояние ПК: «как нагрузка?», «что с памятью?»
- работа с файлами: «найди файл `отчет.docx`», «прочитай `~/Documents/todo.txt`», «создай папку `проекты/новый`»
- PowerShell: «выполни команду `Get-Date`»

### 📝 Память
- факты о тебе: «запомни, что я учусь на JS», «забудь про работу в Яндексе»
- заметки: «запиши идею: сделать сайт на свободном железе», «покажи мои заметки про музыку»
- напоминания: «напомни через 30 минут попить воды», «напомни завтра в 9 утра позвонить маме»

### 🌐 Интернет
- поиск: «найди как починить Wi-Fi на Win10», «что нового в Mistral?»
- погода: «какая завтра погода в Питере?»
- разбор страницы: «суммируй эту статью: <url>»

### 📱 Телефон (Android, через ADB по Wi-Fi)
- подключение: «подключись к телефону 192.168.1.7»
- открыть приложение: «открой YouTube на телефоне», «запусти Telegram на телефоне»
- открыть ссылку: «открой на телефоне github.com»
- скриншот: «скриньни экран телефона»
- ввод: «напечатай на телефоне `привет, мам`»
- клавиши: «нажми home на телефоне», «верни назад»
- батарея: «сколько на телефоне заряда?»
- звонок: «позвони +79…»
- уведомление: «отправь себе на телефон уведомление: не забудь зонт»

### 📊 Привычки и статистика
- Среда сама в фоне (раз в 3 сек) смотрит, какие окна у тебя открыты, агрегирует это в `sreda.db`.
- Анализирует и ищет паттерны: «по утрам открываешь Chrome и YouTube».
- Можешь сказать: «чему ты сегодня научилась?», «покажи мои привычки».
- Внизу левой колонки — графики «День / Неделя / Месяц».

### 🎙 Голос
- Нажми микрофон → говори → автоматически отправится после паузы.
- Кнопка справа от поля включает/выключает озвучку ответов (TTS встроен в Chrome/Edge — без интернета).

## 🌍 Доступ извне (с улицы, другого города)

Локальная сеть — это просто, но за её пределы по умолчанию ничего не выходит. Самый безболезненный способ — **Tailscale**:

1. Установи Tailscale на ПК (Windows) и на телефон.
2. Залогинься одним аккаунтом с обоих устройств.
3. В Tailscale → ПК получит постоянный IP вида `100.X.Y.Z`.
4. С телефона из любой точки заходи по `http://100.X.Y.Z:5000` — будет работать как локально.

Tailscale бесплатен для личного использования и не открывает порт наружу — это p2p VPN. Альтернатива — WireGuard или Cloudflare Tunnel.

- Логи бэка — в консоли запущенного `app.py`. Если что-то падает, ошибка будет там.
- Лог-файл лаунчера — `sreda/data/launcher_errors.log`.
- БД — `sreda/data/sreda.db` (можно открыть SQLite Browser-ом, чтобы посмотреть таблицы `facts`, `notes`, `reminders`, `activity_log`).

### Пайплайн «сказал → сделал»

1. Текст летит по WebSocket в `app.py:handle_user_message`.
2. `brain.process_message` собирает системный промпт (факты, привычки, состояние ПК, время) и историю.
3. Mistral API возвращает JSON: `reply`, `actions[]`, `facts_to_save[]`.
4. `parse_and_execute` сохраняет факты, выполняет actions через диспетчер `execute_action`.
5. Результаты складываются обратно в reply, всё это шлётся в браузер событием `assistant_message`.
6. Если в actions был `web_search`, делается второй проход с результатом поиска в контексте.
7. Если Mistral недоступен — автоматически переключается на локальный Ollama.

### Где живут данные

- **Память (память Среды)** → таблицы `facts`, `notes`, `reminders`, `daily_summary` в `data/sreda.db`.
- **Логи действий Среды** → `action_log`.
- **Логи активности (что было в фокусе)** → `activity_log`, обновляется каждые ~3 сек, через 15 сек агрегируется.
- **Привычки** → `learned_patterns`, пересчитываются перед каждым ответом.
- **Уведомления** → `notifications` (и push в браузер).

## 🔒 Безопасность

- Сервер слушает `0.0.0.0:5000` — то есть открыт для всех в локальной сети. Если у тебя домашняя приватная Wi-Fi сеть — норм. Если делишь Wi-Fi с соседями — поставь Tailscale и поменяй `SERVER_HOST = "127.0.0.1"` в `config.py`, а с телефона ходи через VPN.
- `run_command` выполняет произвольный PowerShell — это мощно и опасно. Среда не запускает команды без явного намерения, но если беспокоишься — закомментируй этот action в `actions.py`.
- Файловые операции (`write_file`, `delete_path`) разрешены везде. Если боишься «удалить лишнее» — Среда подтверждает удаление в чате.
- Mistral API-ключ хранится в `config.py`. Можно вынести в `.env` (`python-dotenv` уже подключён).
